💥什么是炸号

“炸号” 指的是通过举报的方式导致账号异常或者🚫封禁。

🔨精准打击

举报方式一

在推特客户端或者网页，点击帖子右上角的三个点（），然后会显示菜单栏。接着，点击菜单栏里的 “举报帖子” 选项。

举报方式二

在网页版登陆推特账号后，打开 链接，选择举报的理由，然后按照表单📝填写信息提交。

分析内容

在举报之前需要先📈分析推文的内容，选择对应的理由举报才能达到最好的效果。

在搜索栏使用搜索指令语法搜索指定内容。

from:账号ID 关键词

注意，账号 ID 和关键词之间有一个空格。

这样就可以针对指定账户搜索出包含关键词的推文。例如搜索 “死”、“杀”、“捅”、“创”。

搜索出推文后就可以选择 “暴力言论” 为理由。

然后，根据内容的不同，你可以选择 “暴力威胁” 或者 “伤害的愿望”，这两种理由在 “暴力言论” 里是最有效果的。如果是使用举报方式 2，也可以直接打开此 链接。

还可以搜索这些关键词：“操”、“妈”、“逼”、“傻”、“脑”、“婊”、“丑”、“穷”、“病”、“蠢”、“骂”、“屌”、“贱”、“残”。

搜索出推文后选择 “滥用和骚扰” 为理由。

如果这个推文的内容是文字，或者是图文，就选择 “侮辱”；如果是引用其他人的推文再配文字，选择 “有针对性的骚扰”。这样选择效果最佳。如果是使用举报方式 2，也可以直接打开此 链接。

如果是🔞黄推，图片设置了敏感内容，这个时候举报选择 “敏感或令人不安的媒体” 理由其实起不到太大的作用，应该选择 “垃圾信息”，只要累计的举报量够多就会有效果。

举报流程

推特举报机制是有算法的，其中一个考虑因素是举报的数量，另一个是举报内容的严重程度。所以，在进行举报时，优先举报带有🤬辱骂词汇的推文，选择 “滥用和骚扰” 为理由。在累积一定数量的举报后，再搜索推文进行进一步的举报，理由选择 “暴力威胁”。

在累积一定数量的举报后，再提高举报内容的严重程度，这样可以增加举报成功的概率✅。

在举报时，请注意不要在 A 账号完成举报后立即切换到 B 账号再次举报，接着再换到 C 账号继续举报。这样做可能导致自己被🚫shadowban，因为推特会检测举报账号的 IP 地址。

要在不同的时间间隔内使用不同的 IP 进行举报。例如，使用 A 账号在上午通过🇯🇵日本代理进行举报，B 账号在晚上通过🇰🇷韩国代理进行举报，C 账号在第二天通过🇨🇦加拿大代理进行举报，这样才可以避免自己被🚫shadowban。

最好的做法是联合多人，按照以上流程一起举报某一推特号。

以上流程都举报完后，最后再举报一次。理由选择 “儿童安全”，再选择 “诱导或网上引诱未成年人”；背景信息可以自己补充。这样的举报可以使举报的严重程度最大化，加快举报审核的速度，非常有效✅。

🔒锁号怎么办

在常规情况下，锁推或者被拉黑后是无法看到推文内容的，但可以利用搜索引擎的缓存找到曾经被收录的推文链接。

在下面的输入框，输入账号 ID，然后就会显示出这个推特账号发过的推文。把这些推文的链接收集下来，使用举报方式 2 举报推文。即使对方锁推了，按照这种方式举报依然有效✅。

🚼重生后怎么针对

“重生” 指的是一个推主被炸号后又重新注册的小号。

这个其实非常简单，直接到主页举报，就算没有违规行为，只要量够大直接就能封了。因为新账号的权重特别低⬇️，特别是还在🚫shadowban 的期间，账号是非常脆弱的。
